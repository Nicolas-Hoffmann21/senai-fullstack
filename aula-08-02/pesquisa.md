## 1. O que é um objeto em programação? 
 É uma estrutura que agrupa informações (dados) e ações relacionadas a uma única entidade. Ele organiza as informações no código de forma estruturada.
## 2. O que é uma propriedade de um objeto? Cite um exemplo.
 É uma característica ou dado armazenado dentro daquele objeto. Exemplo: a propriedade nome ou idade em um objeto que representa uma pessoa. 
## 3. O que é um método? Qual a diferença entre método e propriedade? 
Um método é uma ação que o objeto sabe executar (uma função interna). A diferença é que a propriedade guarda um dado (característica) e o método executa uma ação (comportamento). 
## 4. O que é uma classe?
 É um molde ou uma estrutura base usada para criar vários objetos parecidos de forma padronizada. 
## 5. Qual a diferença entre classe e objeto?
 A classe é apenas o molde teórico/planta do projeto , enquanto o objeto é o item real criado fisicamente na memória a partir desse molde. 
## 6. Para que serve o constructor dentro de uma classe? 
 Serve para inicializar o objeto e definir os valores iniciais das suas propriedades no momento exato em que ele é criado com o comando new. 
## 7. O que significa a palavra this dentro de uma classe?
 Refere-se ao próprio objeto específico que está executando aquele código ou acessando aquela propriedade naquele momento. 
## 8. O que a palavra new faz ao criar um objeto?
 Ela cria, gera e instancia um novo objeto na memória com base no molde da classe especificada. 
## 9. Em TypeScript, por que usamos tipos como string, number e boolean? 
 Usamos para dar segurança ao código, garantindo que uma variável de texto (string), número (number) ou verdadeiro/falso (boolean) não receba um valor incorreto por engano.
