#**Pesquisa**

1 - O que é programação?
Programação é o processo de dar instruções para um computador executar tarefas específicas. É como escrever uma receita de bolo, só que em uma linguagem que a máquina consegue entender, para criar sistemas, sites ou aplicativos.

2 - O que é um algoritmo?
Um algoritmo é uma sequência passo a passo de instruções lógicas para resolver um problema ou realizar uma tarefa. Ele existe antes do código: o ato de escovar os dentes ou a receita de bolo citada acima são algoritmos da vida real.

3 - Qual a diferença entre algoritmo e linguagem de programação?
O algoritmo é a ideia, a lógica pensada para resolver o problema (independente de computador). A linguagem de programação é a ferramenta e o idioma técnico que você usa para traduzir essa lógica em algo que o computador realmente consiga executar.

4 - O que é JavaScript?
JavaScript é uma linguagem de programação de alto nível, amplamente utilizada para criar interatividade em páginas web. Ela permite que os sites façam muito mais do que apenas exibir textos estáticos, lidando com animações, atualizações de dados em tempo real e comportamentos dinâmicos.

5 - Onde o JavaScript pode ser utilizado atualmente?
Hoje o JavaScript é onipresente. Ele roda no navegador (front-end), em servidores (back-end), na criação de aplicativos para celular (como Android e iOS), em sistemas de automação e até em jogos e internet das coisas (IoT).

6 - O que é o Node.js?
O Node.js não é uma linguagem, mas sim um ambiente de execução para JavaScript. Ele pega o motor que roda o JavaScript dentro do navegador Google Chrome (o V8) e permite que você execute códigos JavaScript direto no seu computador ou servidor.

7 - Qual é a função do Node.js no desenvolvimento com JavaScript?
A principal função do Node.js é permitir que o desenvolvedor use JavaScript para criar o back-end das aplicações. Com ele, você consegue gerenciar bancos de dados, criar APIs, manipular arquivos no sistema operacional e construir servidores web robustos.

8 - que é o VS Code?
O Visual Studio Code (VS Code) é um editor de código gratuito desenvolvido pela Microsoft. Ele é o queridinho dos programadores por ser leve, altamente customizável e ter suporte integrado para ferramentas como Git, terminal e extensões que facilitam a escrita do código.

9 - Qual a diferença entre um editor de texto comum e um editor de código?
Um editor comum (como o Bloco de Notas) serve apenas para texto bruto. Já um editor de código possui ferramentas específicas para programadores, como destaque de cores na sintaxe (syntax highlighting), preenchimento automático de código (IntelliSense) e detecção de erros em tempo real.

10 - O que é o terminal?
O terminal (ou linha de comando) é uma interface de texto onde você digita comandos para interagir diretamente com o sistema operacional do computador. Em vez de clicar em ícones e pastas com o mouse, você faz tudo digitando instruções específicas.

11 - Para que serve o comando node arquivo.js?
Esse comando serve para mandar o Node.js executar o código JavaScript que está guardado dentro do arquivo chamado arquivo.js. Ele roda a lógica do arquivo e exibe os resultados direto na tela do seu terminal.

12 - O que é uma variável?
Uma variável é um espaço na memória do computador que você reserva para guardar algum dado ou informação que seu programa vai precisar usar ou modificar mais tarde. Pense nela como uma caixa etiquetada onde você guarda um valor.

13 - Qual a diferença entre let e const?
A diferença está na capacidade de alteração. Uma variável criada com let pode ter seu valor alterado (mutável) ao longo do código. Já uma variável criada com const define uma constante, ou seja, seu valor inicial não pode ser reatribuído depois.

14 - O que são tipos de dados? Cite exemplos.
Tipos de dados são as classificações que definem a natureza da informação que estamos manipulando. No JavaScript, os principais exemplos são: Number (números como 10 ou 4.5), String (textos entre aspas como "Olá"), Boolean (verdadeiro ou falso) e Object (estruturas complexas).

15 - Qual a principal diferença entre JavaScript e TypeScript?
A principal diferença é a tipagem estática. O JavaScript é de tipagem dinâmica (você muda o tipo da variável quando quiser). O TypeScript adiciona tipos explícitos ao JavaScript, forçando você a dizer se uma variável é estritamente um texto ou número, o que ajuda a pegar erros antes mesmo do código rodar.
